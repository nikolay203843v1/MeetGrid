# Синхрон - Планировщик встреч

Удобный календарь для планирования корпоративных встреч. Делал для того, чтобы быстро раскидывать созвоны по времени без накладок у коллег. 

---

## Что умеет:
* **Удобная сетка часов:** можно переключаться между днем и неделей.
* **Проверка пересечений:** если у кого-то из коллег уже есть встреча на это время, система предупредит, чтобы не было коллизий. При создании встречи организатор может сам себя убрать из списка участников, если нужно.
* **Зум тачпадом:** можно плавно увеличивать или уменьшать высоту часов щипком на тачпаде (или зажав Ctrl + колесико мыши). Масштабирование работает с выбранной анимацией.
* **10 тем оформления и анимаций:** от неонового киберпанка и ретро-вейва до обычного минимализма. Анимации переключения сетки тоже можно менять в настройках (слайды, повороты, глитчи и т.д.).
* **Реалтайм-уведомления:** если тебя добавили во встречу, bell-иконка начинает вибрировать, обновляется счетчик и вылетает аккуратная плашка в углу экрана.
* **Умные подсказки:** при наведении на встречу показывается подробная инфа, карточка автоматически выравнивается по границам экрана и не улетает за край.

---

## Стек:
* **Фронтенд:** чистый JS (ES6+), HTML5, CSS3 (кастомные переменные, гриды, ключевые кадры для анимаций).
* **Бэкенд:** Node.js, Express (простенький REST API).
* **База данных:** SQLite.

---

## Как запустить локально:

Для работы нужен установленный Node.js.

1. Установи зависимости:
   ```bash
   npm install
   ```
2. Запусти сервер:
   ```bash
   npm start
   ```
3. Открой в браузере:
   `http://localhost:3000`
